## 系统概述

本系统采用**双重加密机制**确保数据传输安全：

1. **通信层加密** - 使用共享密钥对传输数据进行AES-256-CBC加密
2. **卡密派生加密** - 基于用户卡密派生唯一密钥，用于本地配置加密

### 加密设计目标
- ✅ 防止中间人攻击窃听通信内容
- ✅ 防止重放攻击（时间戳+签名验证）
- ✅ 确保数据完整性和来源可信
- ✅ 支持离线登录模式（卡密ikun）

## 核心加密模块

### 1. 通信加密模块 (gui/utils/encryptData.py)

```python
# 安全配置 - 与服务器保持一致
SECRET_KEY = 'uno_yyds_wzry_sgs_ljyx'  # 共享密钥
ENCRYPTION_IV = b'1234567890123456'    # 固定IV（16字节）
TIMESTAMP_VALID_WINDOW = 300           # 时间戳有效窗口（5分钟）

class CryptoUtils:
    """加密解密工具类 - 用于客户端与服务器通信"""

    @staticmethod
    def encrypt_data(data: str) -> str:
        """
        加密数据
        
        流程:
        1. 使用SHA256派生32字节密钥
        2. AES-256-CBC加密
        3. Base64编码
        
        Args:
            data: 明文字符串
            
        Returns:
            Base64编码的密文
        """
        key = hashlib.sha256(SECRET_KEY.encode()).digest()
        cipher = AES.new(key, AES.MODE_CBC, ENCRYPTION_IV)
        encrypted_data = cipher.encrypt(pad(data.encode('utf-8'), AES.block_size))
        return base64.b64encode(encrypted_data).decode('utf-8')

    @staticmethod
    def decrypt_data(encrypted_data: str) -> str:
        """
        解密数据
        
        Args:
            encrypted_data: Base64编码的密文
            
        Returns:
            明文字符串
        """
        key = hashlib.sha256(SECRET_KEY.encode()).digest()
        cipher = AES.new(key, AES.MODE_CBC, ENCRYPTION_IV)
        decrypted_data = unpad(
            cipher.decrypt(base64.b64decode(encrypted_data)), 
            AES.block_size
        )
        return decrypted_data.decode('utf-8')

    @staticmethod
    def generate_signature(kami: str, timestamp: int) -> str:
        """
        生成请求签名
        
        签名算法: HMAC-SHA256(kami + timestamp, SECRET_KEY)
        
        Args:
            kami: 卡密字符串
            timestamp: 时间戳
            
        Returns:
            十六进制签名字符串
        """
        data = f"{kami}{timestamp}".encode('utf-8')
        signature = hmac.new(SECRET_KEY.encode(), data, hashlib.sha256).hexdigest()
        return signature
```

### 2. 卡密派生加密模块 (config/kami_encrypt.py)

```python
class KamiEncryptor:
    """
    卡密加密工具类
    基于卡密派生唯一密钥，用于本地配置加密
    """
    
    _GLOBAL_SALT = b"ikun_alliance_2024_salt_v1"

    def _derive_key(self, kami: str) -> bytes:
        """
        从卡密派生加密密钥
        
        密钥派生公式: key = SHA256(kami + salt)
        
        特点:
        - 不同卡密产生完全不同的密钥
        - 无法从密钥反推卡密
        - 每个卡密对应唯一的加密密钥
        
        Args:
            kami: 卡密字符串
            
        Returns:
            32字节密钥
        """
        combined = kami.encode('utf-8') + self._GLOBAL_SALT
        return hashlib.sha256(combined).digest()

    def encrypt(self, data: dict) -> str:
        """
        加密数据
        
        加密格式: base64(IV[16字节] + encrypted_data)
        
        Args:
            data: 要加密的字典
            
        Returns:
            Base64编码的密文
        """
        json_str = json.dumps(data, ensure_ascii=False)
        iv = secrets.token_bytes(16)  # 随机IV
        cipher = AES.new(self._derived_key, AES.MODE_CBC, iv)
        encrypted = cipher.encrypt(pad(json_str.encode('utf-8'), AES.block_size))
        return base64.b64encode(iv + encrypted).decode('utf-8')

    def decrypt(self, encrypted_str: str) -> dict:
        """
        解密数据
        
        Args:
            encrypted_str: Base64编码的密文
            
        Returns:
            解密后的字典
        """
        encrypted_data = base64.b64decode(encrypted_str.encode('utf-8'))
        iv = encrypted_data[:16]
        ciphertext = encrypted_data[16:]
        cipher = AES.new(self._derived_key, AES.MODE_CBC, iv)
        decrypted = unpad(cipher.decrypt(ciphertext), AES.block_size)
        return json.loads(decrypted.decode('utf-8'))
```

### 3. 本地登录数据加密 (gui/utils/jiami.py)

```python
class LoginDataEncryptor:
    """
    登录数据加密器
    使用机器码作为密钥，加密存储登录凭证
    """
    
    def __init__(self):
        self.key = self._get_machine_key()
        self.data_file = config_value.login_data_path

    def _get_machine_key(self) -> bytes:
        """从机器码派生加密密钥"""
        machine_code = get_unique_machine_code()
        return hashlib.sha256(machine_code.encode()).digest()

    def save_login_data(self, data: dict):
        """加密并保存登录数据到本地"""
        json_str = json.dumps(data)
        iv = secrets.token_bytes(16)
        cipher = AES.new(self.key, AES.MODE_CBC, iv)
        encrypted = cipher.encrypt(pad(json_str.encode(), AES.block_size))
        encrypted_str = base64.b64encode(iv + encrypted).decode('utf-8')
        
        with open(self.data_file, 'w') as f:
            f.write(encrypted_str)

    def decrypt(self, encrypted_str: str) -> dict:
        """解密本地存储的登录数据"""
        encrypted_data = base64.b64decode(encrypted_str.encode('utf-8'))
        iv = encrypted_data[:16]
        ciphertext = encrypted_data[16:]
        cipher = AES.new(self.key, AES.MODE_CBC, iv)
        decrypted = unpad(cipher.decrypt(ciphertext), AES.block_size)
        return json.loads(decrypted.decode('utf-8'))
```

```python
"""
卡密加密传输测试工具
用于测试客户端与服务器之间的加密通信
"""

import base64
import hashlib
import hmac
import json
import time
import requests
from Crypto.Cipher import AES
from Crypto.Util.Padding import pad, unpad


# ============ 配置 ============
# 共享密钥 - 必须与服务器保持一致
SECRET_KEY = 'uno_yyds_wzry_sgs_ljyx'
# 固定IV
ENCRYPTION_IV = b'1234567890123456'
# 服务器地址
SERVER_URL = "https://your-server.com/index.php"
# 静态Token
STATIC_TOKEN = "your_static_token"


class KamiCryptoTester:
    """卡密加密传输测试类"""
    
    def __init__(self, kami: str, machine_code: str, version: str = "1.0.0"):
        """
        初始化测试器
        
        Args:
            kami: 测试用卡密
            machine_code: 测试用机器码
            version: 软件版本号
        """
        self.kami = kami
        self.machine_code = machine_code
        self.version = version
        self.key = hashlib.sha256(SECRET_KEY.encode()).digest()
    
    def encrypt_data(self, data: str) -> str:
        """加密数据"""
        cipher = AES.new(self.key, AES.MODE_CBC, ENCRYPTION_IV)
        encrypted = cipher.encrypt(pad(data.encode('utf-8'), AES.block_size))
        return base64.b64encode(encrypted).decode('utf-8')
    
    def decrypt_data(self, encrypted_data: str) -> str:
        """解密数据"""
        cipher = AES.new(self.key, AES.MODE_CBC, ENCRYPTION_IV)
        decrypted = unpad(
            cipher.decrypt(base64.b64decode(encrypted_data)),
            AES.block_size
        )
        return decrypted.decode('utf-8')
    
    def generate_signature(self, timestamp: int) -> str:
        """生成签名"""
        data = f"{self.kami}{timestamp}".encode('utf-8')
        return hmac.new(SECRET_KEY.encode(), data, hashlib.sha256).hexdigest()
    
    def build_request_payload(self) -> dict:
        """
        构建加密请求体
        
        Returns:
            包含加密数据的请求体字典
        """
        timestamp = int(time.time())
        
        # 加密各项数据
        encrypted_kami = self.encrypt_data(self.kami)
        encrypted_machine = self.encrypt_data(self.machine_code)
        encrypted_version = self.encrypt_data(self.version)
        signature = self.generate_signature(timestamp)
        
        payload = {
            'encrypted_kami': encrypted_kami,
            'timestamp': timestamp,
            'signature': signature,
            'encrypted_machine_code': encrypted_machine,
            'encrypted_version': encrypted_version
        }
        
        return payload
    
    def send_login_request(self, timeout: int = 10) -> dict:
        """
        发送登录请求并返回响应
        
        Args:
            timeout: 请求超时时间（秒）
            
        Returns:
            服务器响应字典
        """
        url = f"{SERVER_URL}?static_token={STATIC_TOKEN}"
        payload = self.build_request_payload()
        
        print("=" * 60)
        print("发送登录请求")
        print("=" * 60)
        print(f"\n请求URL: {url}")
        print(f"\n请求体:")
        print(json.dumps(payload, indent=2, ensure_ascii=False))
        
        try:
            response = requests.post(url, data=payload, timeout=timeout)
            response.raise_for_status()
            
            resp_data = response.json()
            print(f"\n响应状态: {response.status_code}")
            print(f"响应内容:")
            print(json.dumps(resp_data, indent=2, ensure_ascii=False))
            
            # 如果包含加密数据，尝试解密
            if 'data' in resp_data and resp_data['data']:
                try:
                    decrypted = self.decrypt_data(resp_data['data'])
                    user_data = json.loads(decrypted)
                    print(f"\n解密后的用户数据:")
                    print(json.dumps(user_data, indent=2, ensure_ascii=False))
                except Exception as e:
                    print(f"\n解密响应数据失败: {e}")
            
            return resp_data
            
        except requests.exceptions.RequestException as e:
            print(f"\n请求失败: {e}")
            return {"code": -1, "msg": f"请求异常: {str(e)}"}
        except json.JSONDecodeError as e:
            print(f"\n解析响应失败: {e}")
            return {"code": -1, "msg": f"JSON解析失败: {str(e)}"}
    
    def verify_signature(self, timestamp: int, signature: str) -> bool:
        """验证签名是否正确"""
        expected = self.generate_signature(timestamp)
        return hmac.compare_digest(expected, signature)
```

## 打包模式选择

### 模式一：卡密登录模式（推荐）

适用于需要授权验证的商业部署场景。

**特点：**
- ✅ 完整的加密传输验证
- ✅ 支持在线/离线登录
- ✅ 卡密有效期管理
- ✅ 设备绑定控制

**配置方式：**
```python
# 在打包配置中启用卡密模式
# build.bat 或打包脚本中设置
set KAMI_MODE=enabled
```

**登录流程：**
1. 用户输入卡密
2. 客户端加密卡密和机器码
3. 发送到服务器验证
4. 返回加密的用户数据
5. 本地存储加密凭证

```python
# 创建免密登录窗口
login_window = LoginWindow(any_kami_mode=True)
```

或在启动时通过命令行参数：
```bash
# 启动免密模式
main.py --free-login
```

### 打包配置说明

在 `build.bat` 中添加模式选择：

```batch
@echo off
chcp 65001

REM ========== 打包模式选择 ==========
echo 请选择打包模式:
echo 1. 卡密登录模式（正式版）
echo 2. 免密登录模式（测试版）
set /p mode="请输入选项 (1/2): "

if "%mode%"=="1" (
    echo 正在打包卡密登录模式...
    set BUILD_ARGS=--include-plugin=gui.utils.encryptData --include-plugin=config.kami_encrypt
) else (
    echo 正在打包免密登录模式...
    set BUILD_ARGS=--free-login-mode
)

REM ========== 执行打包 ==========
python -m nuitka %BUILD_ARGS% main.py
```

## 安全建议

1. **生产环境必须修改共享密钥**
   ```python
   # 修改 gui/utils/encryptData.py 和服务器端的 SECRET_KEY
   SECRET_KEY = 'your_random_long_secret_key_here_32chars'
   ```

2. **使用HTTPS传输**
   - 确保服务器配置了SSL证书
   - 客户端使用https://协议访问

3. **定期更换盐值**
   - 每个版本可以更换 `_GLOBAL_SALT`
   - 需要同步更新服务器端

4. **限制时间戳窗口**
   - 设置合理的时间戳有效窗口（建议5分钟）
   - 防止重放攻击
