# temu-automation-toolkit
基于 Python 的企业级电商自动化系统 | PyQt5 桌面应用 + FastAPI 后端服务 + SQLite 数据库 + 任务调度系统

核心功能：
• 电商运营自动化 - 商品管理、价格调整、活动报名、库存同步
• 智能任务调度 - 定时任务、多线程管理、任务队列、失败重试
• 数据爬取框架 - 代理池管理、会话池、反爬虫策略、多平台扩展
• 财务管理模块 - 账单下载、成本核算、报表生成、数据导出
• 双模式架构 - 桌面 GUI 应用 + RESTful API 服务

技术亮点：异步编程、数据库 WAL 模式、进程守护、日志系统、配置加密
